# Support

## Getting Help

### GitHub Issues (Recommended)

- **Bug reports**: https://github.com/tiagohanna123/yuzer-intelligence-engine/issues
- **Feature requests**: https://github.com/tiagohanna123/yuzer-intelligence-engine/issues

Before opening a new issue, please search existing issues to avoid duplicates.

### Documentation

- **README.md** — Quick start, API reference
- **ARCHITECTURE.md** — Design, data flow, module details
- **LEARN.md** — Learning guide for new developers
- **CONTRIBUTING.md** — How to contribute
- **CHANGELOG.md** — Version history

### Email

For private inquiries or security-related questions, contact the repository owner via GitHub profile.

## Response Times

- **Issues**: typically within 48 hours
- **Feature requests**: reviewed within 1 week
- **Security reports**: see [SECURITY.md](./SECURITY.md)
